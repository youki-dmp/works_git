# AI Talent Production & Automated Operation (AIタレント制作・自動運用プロジェクト)

## 概要
AIアバターを活用した「非属人型YouTubeチャンネル」の構築と、LINEマーケティングによる収益化の完全自動化・半自動化を目指すプロジェクト。

---

## 🚀 5ステップ・プロジェクト計画（PL & MG 監修済）

### Step 1: ニッチ選定とペルソナ構築
*   **内容**: 競合が少なく、かつ単価の高いジャンル（例：シニア向けAI活用）を特定。
*   **成果物**: AIタレント「佐藤 くみこ（54歳）」のビジュアル定義（Nano Banana Pro）、性格設定ガイド、ElevenLabsによる固定ボイス。
*   **佐藤くみこの特徴**: 元生涯学習センター講師。穏やかで知的、シニア層に安心感を与える女性ペルソナ。

### Step 2: 制作パイプラインの構築
*   **内容**: 構成(Gemini 3 Pro) ➔ ビジュアル(Nano Banana Pro) ➔ 映像(Veo 3.1) ➔ 音声(ElevenLabs) ➔ 編集(Remotion) のワークフローを構築。
*   **状況**: シーン別の映像生成およびRemotionテンプレートの実装を完了。


### Step 3: 最小構成（MVP）リリース
*   **内容**: 「佐藤久美子」による、シニア向けAI活用術の解説動画（ショート3本、長尺1本）を制作。
*   **構成案**:
    1.  挨拶・理念紹介（今回生成したシーン）
    2.  スマートフォンの「魔法の道具化」具体例（画像生成など）
    3.  LINE特典（PDF教材）への誘導
*   **目的**: YouTubeアルゴリズムとの相性と、AIアバターに対する視聴者の反応（信頼感・親しみやすさ）を検証。

### Step 4: LINEファンネル連携
*   **内容**: 動画から公式LINEへ誘導。Clawdbotのメッセージ機能を使い、登録者へ無料特典「シニアのためのAI活用・初歩の初歩ガイド（PDF）」を自動配布。
*   **目的**: YouTubeの広告収益に依存しない、独自のリストビジネスを確立。

### Step 5: コスト最適化と完全自動スケール
*   **内容**: 1本あたりの制作コストと収益（ROI）を算出。黒字化を確認後、投稿頻度を上げ、多プラットフォーム（TikTok, Instagram）へ展開。
