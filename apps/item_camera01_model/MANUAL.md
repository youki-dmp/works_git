# 動作マニュアル / Operating Manual - item_camera01_model

1枚のPNG画像を使用して、WebカメラとマイクでVTuberのように配信するためのアプリケーションです。

## 🚀 使い方 / How to Use

### 1. サーバーの起動 / Start Server
このディレクトリでローカルサーバーを起動してください。
```bash
python3 -m http.server 8000
```
その後、ブラウザで `http://localhost:8000` にアクセスします。

### 2. アプリの初期設定 / Setup
- **イラストの選択**: 「ファイルを選択」ボタンからPNG画像をアップロードします。
  - サンプルの狐耳キャラは `assets/sample.png` にあります。
- **感度調整**: 「Volume Sensitivity」スライダーで、声に対する反応の大きさを調整できます。
- **開始**: 「Start App」ボタンを押して、カメラとマイクの権限を許可してください。

- **4枚の別画像モード**:
  - 「通常 / 目閉じ / 口開け / 両方」の画像を、それぞれ**別のファイルとしてアップロード**します。
  - スプライトシートを作るのが面倒な場合や、配置がズレる場合に最適です。
  - **「モード選択」** で「4枚の別画像」を選んでください。

### 4. 位置合わせ / Calibration (重要)
- 「アプリを開始」した後、赤い **「位置合わせを実行」** を押してください。これであなたの顔の向きが同期されます。

### 4. 配信での利用 (OBS) / OBS Integration
- OBSで「ブラウザソース」を追加します。
- URLに `http://localhost:8000` を入力し、背景が透過されることを確認してください。

## 🛠️ 機能紹介 / Features
- **顔限定メッシュ**: 画像全体を歪ませるのではなく、顔のパーツ（目、鼻、口）がある範囲だけをピンポイントで変形させます。これにより、服や背景が動いてしまうのを防ぎます。
- **日本語インターフェース**: すべてのコントロールが日本語化されています。

## ⚠️ 注意事項 / Notes
- 極端に横を向くと、メッシュが崩れる場合があります。正面付近での利用を推奨します。
- 1枚絵の変形のため、目を完全に閉じても「閉じ目のまつ毛」が描き足されるわけではありません（元の絵が潰れる表現になります）。
