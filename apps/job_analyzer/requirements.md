# 私の希望勤務条件 (My Job Requirements)

このファイルにあなたの希望条件を書いてください。AIはこの内容と求人情報を比較します。

## 1. 基本条件 (Must Haves)
絶対に譲れない条件を書いてください。
- **年収**: 例）最低 600万円以上希望
- **勤務地**: 例）東京23区内、またはフルリモート
- **リモートワーク**: 例）週3日以上はリモート必須
- **雇用形態**: 例）正社員のみ

## 2. 歓迎条件 (Nice to Haves)
あると嬉しい条件を書いてください。
- **福利厚生**: 例）書籍購入補助、勉強会参加費補助
- **勤務時間**: 例）フレックスタイム制（コアタイムなしが望ましい）
- **技術スタック**: 例）Python, Go, React, AWS, GCP のいずれかを使用していること

## 3. NG条件 (Deal Breakers)
これがあったら絶対に応募しないという条件を書いてください。
- **技術**: 例）COBOL, VBAメインの案件
- **環境**: 例）スーツ着用必須、客先常駐（SES）
- **その他**: 例）みなし残業45時間以上

## 4. キャリアの方向性
どのようなスキルを伸ばしたいか、自由記述で書いてください。
- 例）バックエンドのパフォーマンスチューニングに強くなりたい。
- 例）マネジメントよりはスペシャリストとしてコードを書き続けたい。
