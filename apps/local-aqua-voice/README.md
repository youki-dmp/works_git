# Voice Input-kun / ボイス入力くん

A high-accuracy, local Japanese voice input application for macOS.
macOS向けの高精度なローカル日本語音声入力アプリです。

## Download / ダウンロード
- [**VoiceInputKun_macOS.zip (ビルド済みapp版)**](./dist/VoiceInputKun_macOS.zip?raw=true)
  - ダウンロードして解凍後、「ボイス入力くん.app」をアプリケーションフォルダに入れて使用してください。

Refer to [MANUAL.md](./MANUAL.md) for usage and installation details.
詳細な使用方法やインストール手順については [MANUAL.md](./MANUAL.md) を参照してください。
