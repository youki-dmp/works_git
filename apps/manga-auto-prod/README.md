# 4-Panel Manga Auto Production Project

## 概要
Google AntigravityとNano Banana Pro（Gemini 3 Pro Image）を活用し、キャラクターの一貫性を保った4コマ漫画を自動生成するプロジェクトです。

## 技術スタック
- **LLM (Antigravity)**: プロット作成、コマごとのプロンプト生成。
- **Image Gen (Nano Banana Pro)**: 4枚の画像を生成。
- **Layout (Canvas/Python)**: 生成した画像を4コマ形式に統合。
