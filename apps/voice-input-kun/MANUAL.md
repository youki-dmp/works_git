# ボイス入力くん – ユーザーマニュアル

## 概要
ローカルで動作する高精度日本語音声入力アプリです。メニューバーに常駐し、ショートカットキーで録音・文字起こしを行い、テキストをアクティブアプリに自動入力します。

## 起動方法（推奨）

ターミナルを開き、以下のコマンドを **一度だけ** 実行して設定を完了させてください。次から `voice` と打つだけでアプリが起動できるようになります。

```bash
# 設定（初回のみ） ※ ●●をあなたのユーザー名から、フォルダの保存場所を指定して書き換えてください
echo "alias voice='/Users/●●/voice-input-kun/dist/ボイス入力くん.app/Contents/MacOS/ボイス入力くん'" >> ~/.zshrc
source ~/.zshrc

# 起動
voice
```

## 注意事項: セキュリティの許可
アプリを初めて使用する際は、macOSの **システム設定 > プライバシーとセキュリティ** で以下の許可を与えてください。
- **マイク**: 録音のために必要です。
- **アクセシビリティ**: 変換したテキストを自動入力するために必要です。「ターミナル」のスイッチを **ON** にしてください。


## 操作方法

### アプリの状態
| 待機中（Ready） | 録音中（Recording） |
|:---:|:---:|
| ![Ready](./docs/images/ready.png) | ![Recording](./docs/images/recording.png) |

| 操作 | 説明 |
|------|------|
| **ショートカットキー** | デフォルトは `Command + .`（ドット）。このキーで録音開始/停止をトグルします。 |
| **メニュー** | アイコンを右クリックするとメニューが表示されます。<br>・`Configure Shortcut` – ショートカットを変更<br>・`Quit` – アプリ終了 |
| **録音** | ショートカットで録音開始 → 再度ショートカットで停止 → 文字起こしが開始され、結果がアクティブウィンドウに自動入力されます。 |

## ショートカットキーの変更方法
1. メニューバーアイコンを右クリック → **Configure Shortcut** を選択
2. 表示されるダイアログに新しいショートカット文字列を入力（例: `<cmd>+shift+v`）
3. **OK** を押すと設定が保存され、即座に新しいショートカットが有効になります。
